## ⚠️ Nota Importante

Estos archivos se encuentran en `.docs-private/` que está en `.gitignore`. Esto significa:

✓ Los archivos se mantienen localmente en tu computadora
✓ No se sincroniza con GitHub (privados)
✓ No aparecen en el repositorio público
✓ Accesibles solo localmente

Para acceder a ellos desde tu máquina:
```bash
# Navegar a documentación privada
cd .docs-private
ls -la

# Ver un archivo específico
cat SECURITY.md
# o
code SECURITY.md
```
